## Introducción
Este proyecto consiste en una simulación de tráfico urbano desarrollada en Unity, donde varios vehículos circulan por una intersección controlada mediante un sistema de semáforos inteligentes.  
La simulación utiliza sensores y un controlador de tráfico para priorizar el carril más congestionado, optimizando el flujo vehicular. 

---

## ¿Qué incluye el proyecto?
El repositorio contiene:
- El proyecto Unity con la escena de simulación.  
- Scripts que implementan la lógica de vehículos, sensores y semáforos.  
- Controlador de tráfico que calcula la prioridad de cada carril.  
- Estructura base de *waypoints* utilizada para la navegación de los coches.  
- README con información de uso y documentación.

---

## Objetivo del proyecto
El propósito es diseñar una arquitectura clara y modular para una simulación de tráfico, integrando elementos de física simple, IA simbólica y patrones de diseño. 

---

## Origen de la simulación
Como base inicial se utiliza la estructura de *waypoints* del repositorio open-source  
**unity-traffic-simulation**, que fue extendida y modificada para incorporar:
- Sensores funcionales por carril.  
- Controlador de semáforos inteligente.  
- Modelos de comportamiento vehicular modificados.  
- Nuevo flujo de simulación más acorde al objetivo académico.

Repositorio original: https://github.com/mchrbn/unity-traffic-simulation
